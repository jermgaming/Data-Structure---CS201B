1. **Stack push algorithm**: The `push` function checks if the stack is full (i.e., `top` is equal to `size-1`). If it is, it prints an error message and returns. Otherwise, it increments the `top` variable and stores the element at the new `top` index in the array.
2. **Stack pop algorithm**: The `pop` function checks if the stack is empty (i.e., `top` is equal to `-1`). If it is, it prints an error message and returns. Otherwise, it prints the top element and decrements the `top` variable.
3. **Stack peek algorithm**: The `peek` function checks if the stack is empty (i.e., `top` is equal to `-1`). If it is, it prints an error message and returns. Otherwise, it prints the top element.
4. **Stack display algorithm**: The `display` function checks if the stack is empty (i.e., `top` is equal to `-1`). If it is, it prints an error message and returns. Otherwise, it prints all elements in the stack by iterating from the top element to the bottom element.
