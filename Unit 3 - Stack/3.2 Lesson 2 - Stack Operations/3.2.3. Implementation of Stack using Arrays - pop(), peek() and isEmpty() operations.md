### Algorithm Explanation
The program uses the following algorithm to implement the stack operations:
1. The `push()` function checks if the stack is full by comparing the `top` index with the maximum size of the array. If the stack is full, it prints an "overflow" message and returns.
2. If the stack is not full, the `push()` function increments the `top` index and stores the new element at the corresponding index in the `arr` array.
3. The `pop()` function checks if the stack is empty by comparing the `top` index with -1. If the stack is empty, it prints an "underflow" message and returns.
4. If the stack is not empty, the `pop()` function decrements the `top` index and prints the value of the top element.
5. The `peek()` function checks if the stack is empty by comparing the `top` index with -1. If the stack is empty, it prints an "underflow" message and returns.
6. If the stack is not empty, the `peek()` function prints the value of the top element without modifying the `top` index.
7. The `isEmpty()` function checks if the stack is empty by comparing the `top` index with -1. If the stack is empty, it prints an "empty" message.
8. The `display()` function checks if the stack is empty by comparing the `top` index with -1. If the stack is empty, it prints an "empty" message and returns.
9. If the stack is not empty, the `display()` function prints all elements in the stack by iterating over the `arr` array from the `top` index down to 0.
