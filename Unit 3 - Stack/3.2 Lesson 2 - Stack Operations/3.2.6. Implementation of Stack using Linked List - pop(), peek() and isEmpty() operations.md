### Functions Documentation
- **Function Name**: `pop()`
  - **Purpose**: Removes the top element from the stack and returns the new top element.
  - **Parameters**: None
  - **Returns**: The new top element (or NULL if the stack is empty)
  - **Logic**: Checks if the stack is empty, saves the value of the top element, updates the `top` pointer, frees the memory allocated for the removed node, and returns the new top element.
- **Function Name**: `peek()`
  - **Purpose**: Returns the top element of the stack without removing it.
  - **Parameters**: None
  - **Returns**: None (prints the top element to the console)
  - **Logic**: Checks if the stack is empty and prints the value of the top element if it's not.
- **Function Name**: `isEmpty()`
  - **Purpose**: Checks if the stack is empty.
  - **Parameters**: None
  - **Returns**: None (prints a message to the console indicating whether the stack is empty)
  - **Logic**: Checks if the `top` pointer is NULL and prints a message accordingly.
- **Function Name**: `push(int x)`
  - **Purpose**: Adds a new element to the top of the stack.
  - **Parameters**: `x` (the value to be added to the stack)
  - **Returns**: The new top element (or NULL if the stack is full)
  - **Logic**: Allocates memory for a new node, sets the value of the new node, updates the `next` pointer of the new node, and updates the `top` pointer.
- **Function Name**: `display()`
  - **Purpose**: Prints the elements of the stack to the console.
  - **Parameters**: None
  - **Returns**: None
  - **Logic**: Traverses the linked list and prints the value of each node.
