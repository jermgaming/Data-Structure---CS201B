### Algorithm Explanation
The program uses a simple iterative approach to read and display the array elements. Here is a step-by-step breakdown of the main algorithm:
1. Declare an integer array `a` with a fixed size of 10 elements, an integer variable `i` for indexing, and an integer variable `n` to store the number of values to read.
2. Prompt the user to enter the number of values they want to read using `printf`.
3. Read the user's input using `scanf` and store it in the variable `n`.
4. Use a `for` loop to iterate over the array elements, starting from index 0 and going up to `n-1`.
5. Inside the loop, prompt the user to enter the value of each array element using `printf`.
6. Read the user's input using `scanf` and store it in the corresponding array element using the expression `a + i`, which is equivalent to `&a[i]`.
7. After reading all the input values, prompt the user that it will display the array elements.
8. Use another `for` loop to iterate over the array elements, starting from index 0 and going up to `n-1`.
9. Inside the loop, display the value of each array element using `printf` and the expression `*(a + i)`, which is equivalent to `a[i]`.
