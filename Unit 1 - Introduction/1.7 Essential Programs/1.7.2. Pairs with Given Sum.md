### **Step 3: Read the Array Size**

### **Step 4: Create an Array**
```c

```
This creates an array (a list) that can hold `n` numbers. 

**Think of it like:** Creating `n` boxes to store numbers.

### **Step 5: Fill the Array**

```c
    scanf("%d",&arr[i]);
```

This loop runs `n` times and fills each box with a number.

**Example:**

### **Step 6: Read the Target Sum**
```c

```
This reads the target sum you're looking for.

**Example:** If you enter `10`, the program will find all pairs that add up to 10.

---

### **Step 7: Initialize Counter**
```c
int count=0;
```
This variable keeps track of how many pairs we find. We start at 0.

---

### **Step 8: The Main Logic - Nested Loops**
```c
for(int i=0;i<n;i++){
    for(int j=i+1;j<n;j++){
        if(arr[i]+arr[j]==s){
            printf("%d %d\n",arr[i],arr[j]);
            count++;
        }
    }
}
```

This is the heart of the program! Let's break it down:

#### **Outer Loop (i loop):**
- Picks the first number of the pair
- Runs from index 0 to n-1

#### **Inner Loop (j loop):**
- Picks the second number of the pair
- Starts from `i+1` (so we don't pick the same number twice or repeat pairs)
- Runs till the end of the array

#### **Why `j=i+1`?**
To avoid:
1. Using the same element twice (like arr[2] + arr[2])
2. Printing duplicate pairs (like (2,8) and (8,2))

## ğŸ“¤ Output Format

The program prints:
* **Each pair of numbers** that sum up to the target. Each pair is printed on a new line in the format `"a b"`, where `a` and `b` are the elements of the pair.
* **After printing all pairs**, the total count of such pairs is printed on a new line.

**Example Output Structure:**
```
a1 b1
a2 b2
a3 b3
count
```

Where each line `ai bi` represents a pair that sums to the target, and `count` is the total number of pairs found.

**O(nÂ²)** - This means if you have `n` elements, the program does roughly `n Ã— n` operations.

**Why?** Because we use two nested loops:
- Outer loop runs `n` times
- Inner loop runs (n-1), (n-2), (n-3)... times

**For n=5:** Approximately 5 Ã— 4 / 2 = 10 comparisons
