1. Declare a two-dimensional array `a` with a maximum size of 5x5.
2. Prompt the user to input the row and column sizes using `printf()`.

4. Use two nested loops to iterate over the rows and columns of the matrix. The outer loop iterates over the rows, and the inner loop iterates over the columns.
5. Inside the inner loop, prompt the user to input the value of each matrix element using `printf()`.
6. Read the user-inputted value using `scanf()` and store it in the corresponding element of the array `a`.
7. After all matrix elements have been inputted, print a message indicating that the entered matrix will be displayed.
8. Use another pair of nested loops to iterate over the rows and columns of the matrix and print each element using `printf()`.
