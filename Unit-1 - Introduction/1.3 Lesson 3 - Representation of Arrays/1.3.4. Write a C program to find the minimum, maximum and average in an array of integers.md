3. **Calculation**: The program iterates through the array using a `for` loop, performing the following operations:

   * It checks if the current element is less than the current `min` value. If so, it updates the `min` value.

4. **Average Calculation**: After the loop completes, the program calculates the average value by dividing the `avg` variable by the `number` of elements.
5. **Output**: The program prints the minimum, maximum, and average values to the console using `printf()`.
