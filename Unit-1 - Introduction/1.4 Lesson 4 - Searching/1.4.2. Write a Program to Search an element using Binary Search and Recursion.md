## üîß Implementation Details

### Algorithm Explanation
The binary search algorithm works by repeatedly dividing the search interval in half. If the value of the search key is less than the item in the middle of the interval, the next interval will be the lower half. Otherwise, the next interval will be the upper half. The process is repeated until the size of the interval is zero, at which point the search key is not in the array.

### Functions Documentation
- **Function Name**: `void read1(int arr[], int size)`
  - **Purpose**: Reads input from the user and stores it in the array.
  - **Parameters**: 
    - `arr`: The array to store the input elements.
    - `size`: The number of elements to read from the user.
  - **Returns**: None (void).
  - **Logic**: The function uses a for loop to iterate `size` times, prompting the user to enter an element each time and storing it in the corresponding index of the `arr` array.
- **Function Name**: `void bubbleSort(int arr[], int size)`
  - **Purpose**: Sorts the array in ascending order using the bubble sort algorithm.
  - **Parameters**: 
    - `arr`: The array to sort.
    - `size`: The number of elements in the array.
  - **Returns**: None (void).
  - **Logic**: The function uses nested for loops to compare each pair of adjacent elements in the array. If the elements are in the wrong order, they are swapped. This process is repeated until no more swaps are needed, indicating that the array is sorted.
- **Function Name**: `void display(int arr[], int size)`
  - **Purpose**: Displays the elements of the array.
  - **Parameters**: 
    - `arr`: The array to display.
    - `size`: The number of elements in the array.
  - **Returns**: None (void).
  - **Logic**: The function uses a for loop to iterate over the array, printing each element.
- **Function Name**: `int binarySearch(int arr[], int start, int end, int key)`
  - **Purpose**: Searches for the `key` element in the sorted array using the binary search algorithm and recursion.
  - **Parameters**: 
    - `arr`: The sorted array to search.
    - `start`: The starting index of the search range.
    - `end`: The ending index of the search range.
    - `key`: The element to search for.
  - **Returns**: The index of the `key` element if found, -1 otherwise.
  - **Logic**: The function calculates the middle index of the current search range. If the middle element matches the `key`, its index is returned. If the `key` is less than the middle element, the function recursively searches the lower half of the range. If the `key` is greater than the middle element, the function recursively searches the upper half of the range. If the search range becomes empty, the function returns -1 to indicate that the `key` is not in the array.
