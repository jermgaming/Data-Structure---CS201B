The primary data structure used in this program is a singly linked list (SLL). A singly linked list is a linear data structure in which each element is a separate object, and each element (or "node") points to the next node in the sequence. This structure allows for efficient insertion and deletion of nodes at any position in the list.

A node in the linked list is defined as follows:
```c
struct node {
    int data;
    struct node *next;
};
typedef struct node *NODE;
```
Each node has an integer `data` field to store the node's value and a `next` field to point to the next node in the list.
